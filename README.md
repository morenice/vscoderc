# extensions
* Launch VS Code.
* Open the Command Palette (⇧⌘P)
* Typing
```
shell command: Install 'code' command in PATH
```

Install vscode extensions
```
$ ./install_extensions.sh
```

Make extensions file
```
code --list-extensions > extensions
```
